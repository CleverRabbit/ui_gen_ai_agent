# AI-Агент: Эргономичные формы 1С

Веб-приложение для генерации улучшенных интерфейсов форм 1С на основе скриншота и текстового описания.

## Возможности

- 📸 **Анализ скриншота**: Загрузите скриншот существующей формы 1С
- 📝 **Описание задачи**: Опишите проблемы и требования к улучшению
- 🤖 **AI-анализ**: Google Gemini анализирует элементы, стиль и эргономику
- 🎨 **Визуальный макет**: Pollinations.ai генерирует улучшенный дизайн формы
- 💻 **Код 1С**: Генерация рабочих обработчиков событий для модуля формы

## Технологии

- **Frontend**: Streamlit (Python)
- **AI Analysis**: Google Gemini 1.5 Flash API
- **Image Generation**: Pollinations.ai API
- **Все сервисы бесплатные**

## Быстрый старт

### Локальный запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
streamlit run app.py
```

Приложение откроется по адресу: http://localhost:8501

### Развёртывание в Streamlit Cloud

1. Зарегистрируйтесь на [share.streamlit.io](https://share.streamlit.io)
2. Подключите ваш GitHub репозиторий
3. Укажите:
   - Main file path: `app.py`
   - Python version: 3.10+
4. Нажмите "Deploy"

После развёртывания вы получите публичную ссылку на приложение.

## Как использовать

1. **Загрузите скриншот** формы 1С (PNG, JPG, JPEG)
2. **Введите описание**:
   - Какая задача решается формой
   - Что неудобно в текущем варианте
   - Что хотелось бы улучшить
3. **Нажмите "Сгенерировать форму"**
4. **Получите результаты**:
   - 🔍 Анализ элементов и проблем эргономики
   - 🎨 Визуальный макет улучшенной формы
   - 💻 Код обработчиков событий для 1С
